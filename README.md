# 仕様
- スプラッシュ画面を作る
- トップは画像を複数枚でスライドショー
- ヘッダーのアイコンを押すとそこまでスクロール
- スクロールすると要素がフェードインしてくる
- スクロールすると背景は遅めにスクロール(orできなければ固定)
